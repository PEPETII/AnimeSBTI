一个纯前端的动漫人格测试网页项目。

- 300 道情境题
- 81 个动漫角色
- 4 个核心维度组合判定

## 本地运行

这是静态网页项目，无需安装依赖。

1. 直接双击打开 `index.html`。
2. 或使用任意静态服务器在项目根目录启动后访问。
